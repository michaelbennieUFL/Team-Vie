# Team-Vie

## 3D Interactive Website

An interactive 3D website featuring "Team Vie" in animated 3D text with game-like controls.

### Features

- **3D Animated Text**: "TEAM VIE" rendered in 3D with continuous animations
- **Interactive Controls**: 
  - Move your mouse to rotate the view
  - Click on letters to interact and score points
  - Scroll to zoom in/out
- **Visual Effects**:
  - Particle system background
  - Dynamic lighting with multiple colored lights
  - Click explosion effects
  - Smooth hover animations
- **Game Elements**:
  - Score tracking system
  - Interactive letter clicking
  - Real-time animations

### How to Use

1. Open `index.html` in a web browser
2. Move your mouse around to control the view
3. Click on individual letters to interact and gain points
4. Use the scroll wheel to zoom in and out

### Technologies Used

- CSS 3D Transforms for 3D rendering
- HTML5 for structure
- Pure JavaScript for interactivity
